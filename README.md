# 🏦 CreditWise Loan Approval Prediction System

CreditWise is a Machine Learning-based web application that predicts whether a loan application will be approved or rejected based on applicant financial, credit, and employment details.

The project uses a Logistic Regression classification model trained on 27 engineered features, including income, loan details, employment status, credit score, and derived risk metrics such as DTI ratio squared.

The model is deployed using Streamlit, providing an interactive and user-friendly interface where users can enter applicant details and receive real-time approval predictions along with probability scores.

This project demonstrates end-to-end ML workflow:
- Data preprocessing & feature engineering  
- Model training & evaluation  
- Model serialization  
- Deployment with Streamlit  

Built with Python, Scikit-learn, NumPy, and Streamlit.
